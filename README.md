# **A-Dong 아주대학교 동아리 관리 웹시스템**

> Front-end 브랜치 입니다.

## Test 방법

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```

## Git 사용법
``` bash
# 내 local의 front-end 브랜치에서 remote(Github)의 front-end 브랜치에 push 하는 방법
git push origin front-end:front-end
```
