## Git 사용법
``` bash
# github에서 front-end 브랜치 clone 받는 방법
git clone -b server https://github.com/jjw8967/WebSystem_Project.git

# 내 local의 front-end 브랜치에서 remote(Github)의 front-end 브랜치에 push 하는 방법
git push origin server:server
```
